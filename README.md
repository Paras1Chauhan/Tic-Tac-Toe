## 📖 About

Classic Tic Tac Toe with 2-player mode, automatic win detection and a clean responsive UI — built with JavaScript, HTML & CSS

## ✨ Features

- 2-player local multiplayer
- Automatic win & draw detection
- Score tracking across rounds
- Responsive clean UI
- Restart & reset functionality

## 🛠️ Tech Stack

- **JavaScript**
- **HTML**
- **CSS**

## 🚀 Getting Started

```bash
git clone https://github.com/Paras1Chauhan/Tic-Tac-Toe.git
cd Tic-Tac-Toe
open index.html in your browser
```
